# OpenClaw Agent Modes

## Overview

This framework helps you identify how you're using AI agents and whether you're growing or declining cognitively.

```
┌─────────────────────────────────────────────────────────┐
│                   AI AGENT USAGE                        │
├─────────────────┬───────────────────┬───────────────────┤
│    CAPTAIN      │    ARCHITECT      │    ABDICATOR      │
│    将才         │     帅才          │      刘禅         │
├─────────────────┼───────────────────┼───────────────────┤
│  Learn & Do     │  Design & Verify  │  Accept & Forget  │
│  Growth ↑       │  Efficiency ↑     │  Decline ↓        │
└─────────────────┴───────────────────┴───────────────────┘
```

## The Core Question

> **Am I Captain, Architect, or Abdicator right now?**

## Quick Decision Tree

```
Did you delegate to AI?
    │
    ├─ No → You're not using AI yet
    │
    └─ Yes → Do you understand how AI solved it?
              │
              ├─ No → ABDICATOR ⚠️
              │        You're being used by AI
              │
              └─ Yes → Did you learn something new?
                        │
                        ├─ Yes → CAPTAIN ✓
                        │         You're growing
                        │
                        └─ No → ARCHITECT ✓
                                  You're efficient
```

## Mode Comparison

| Aspect | Captain 将才 | Architect 帅才 | Abdicator 刘禅 |
|--------|-------------|---------------|----------------|
| **Role** | Works alongside | Designs systems | Passes off |
| **Focus** | Learning | Efficiency | Convenience |
| **Verification** | Constant | Checkpoints | None |
| **Outcome** | Growth | Scale | Decline |
| **Cognitive** | Active | Active | Passive |
| **Risk** | Burnout | Over-systematizing | Cognitive debt |

## Practical Applications

### For OpenClaw Users

**Captain Mode:**
```
Task: "Help me write a Python script"
Action: Watch how AI approaches it
Result: Learn Python patterns + get script
```

**Architect Mode:**
```
Task: "Automate daily reporting"
Action: Design workflow, verify at each step
Result: Reliable automation system
```

**Abdicator Mode (AVOID):**
```
Task: "Do my work"
Action: Accept whatever comes back
Result: No learning, cognitive decline
```

## The MIT Study Context

The "Cognitive Debt" paper showed heavy AI users have weakened brain connectivity.

**But:** The data shows *passive consumption* weakens cognition, not active leadership.

> AI didn't make anyone dumber. Giving up thinking makes people dumber.
